# Security Concepts on Campus
## [UWA Security](https://www.uwa.edu.au/about/campus-services/security-and-emergencies/security-services)
UWA has on-campus security (known as UWA Security) who offer a few different services for students (as well as staff and other visitors) depending on their needs. Using the SafeZone app (a third-party app used for security purposes by several organisations), you are able to connect directly with UWA Security at any time of day. UWA Security also conducts 24/7 patrolling on campus. At various points on campus (mostly areas expected to have student coverage 24/7 like libraries), there are also help points which have a button to connect directly with UWA Security. Finally, UWA Security also provide an escort service which allows anyone on campus to be escorted by a security officer from their location to another location on campus (like car parks, College Row, other buildings, etc). An SMS communication service is also available for when security is needed but verbal communication isn't possible.

## [Information Security](https://www.education.gov.au/download/10549/higher-education-providers-2020-mission-based-compacts/20224/university-western-australia-compact-2020/pdf)
UWA has an information security management framework based on the ISO 27001 standard which defines mechanisms for how it handles information security (specifically making reference to protection and detection of cyberattacks) and mitigates the risk of cyberattacks. It is claimed that such protections are driven by UWA IT's Cyber Security and Technology Risk function, with Human Resources providing personnel security and Campus Management ensuring physical security of IT assets.

## [The Security Management System (SMS)](https://www.uwa.edu.au/about/-/media/project/uwa/uwa/about/docs/campus-management/facilities-management/f-security-services-2023.pdf)
Outside of these specific types of security, UWA implements some additional, more general security measures which are typical of most organisations. These include:
- Closed-Circuit Television (CCTV) Systems
- Electronic Access Control (EAC) Systems

among other security measures which are too in-depth to cover here. These security mechanisms are all connected to a LAN (communicating over IP) specifically for these devices, allowing them to communicate with each other (and provide UWA Security with information when necessary, like the camera feed of a building where an alarm is going off). Most (if not all) of the security mechanisms used by UWA follow established standards, whether that be standards developed by Standards Australia and Standards New Zealand (AS/NZS), the International Electrotechnical Commission (IEC), the International Organization for Standardization (ISO), or the National Construction Code (NCC).

CCTV is a common security measure taken by organisations. At UWA, this camera system is known as the Video Surveillance System (or VSS) which consists of several pieces of equipment like fixed cameras, pan-tilt-zoom cameras (of which there are a limited number), network video recorders (which retain footage for a minimum of 35 days), display monitors, etc. The VSS allows for surveillance of campus, specifically being designed to assist in "identifying [people] at key access points, verifying alarms, detecting suspicious behaviour and assisting in post incident investigations". In 2020 it was claimed that the campus "has very limited and inconsistent external [CCTV] coverage". The specific areas covered include vehicle entry/exit points, external areas of buildings, building entry/exit points, and high risk/value areas. Although the VSS does not specifically cover lecture theatres, it does cover all areas that are accessible by students 24/7 with general surveillance.

The Electronic Access Control system is implemented all over campus using keycards. Specifically, this system is set up using Gallagher's security products using the [T15 Multi Tech Reader](https://products.security.gallagher.com/security/au/en_AU/products/access-control-hardware/readers-and-terminals/t15-multi-tech-reader/p/BVC300480) to facilitate this control. Authorised people (mostly consisting of staff and students) are given keycards (specifically [MIFARE DESfire EV3](https://products.security.gallagher.com/security/au/en_AU/products/access-control-hardware/credentials/mifare-desfire/p/C297473) cards), which allow them to access certain aspects of campus that are closed to the general public. This includes things like most buildings after hours, the libraries during exam periods, and other similar areas which the general public cannot access. Additionally, according to the 2023 UWA Security Standards, access control systems are required for sensitive areas and hazardous areas. In some cases, wireless access control systems are used on campus (where there are multiple individual spaces which require higher credential control and wired EAC would be costly). These use something like the [Salto XS4 Mini](https://saltosystems.com/en-au/products/electronic-locks/xs4-mini/) (exact model not specified), which integrate with the existing SMS infrastructure.
