## Padlocks
These are a type of handheld lock which are not permanently fixed to anything, but rather have the ability to be removed for all sorts of use cases. Padlocks are probably the most common and most recognisable type of lock.

## Mortise Locks
Mortise locks are a common type of lock where the lock is [built into a recessed pocket of the door](https://johnbarnesgroup.au/what-is-a-mortice-lock-when-and-where-to-use-them/) rather than being attached to the door's surface. They [require a key to be inserted](https://kerisys.com/resources/what-is-a-mortise-lock-and-the-benefits-of-using-one/) before they open, ensuring that only authorised people can access what is beyond the door. There are various types of mortise locks, including deadlocks.

## Kensington Lock
The Kensington lock is a type of lock used on portable electronic devices to keep them fixed to a stationary object with a cable (similar to a bike lock). They act as anti-theft mechanisms, but are not intended to be used as a permanent locking tool.
