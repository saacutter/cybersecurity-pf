## [Case 1](https://chatgpt.com/share/699d0a67-db5c-800b-ac8d-b5f1ab7e6fd6)
To identify the first case of bias, I used the image generation tool provided on the web client for ChatGPT (which uses the DALL-E 3 model combined with GPT-5.2). The specific purpose of this was to identify biases in the image generation models. To do this, I asked it to generate an image of a doctor and a nurse using the following prompt:
> Generate an image of a doctor and a nurse, making it clear which is which.

There are a few reasons why I chose to do this. However, the main reason was to identify any gender biases in the model's training data. This is a classic example of gender bias, where doctors are often seen as male and nurses as female (even though there are several examples of the contrary to both of these) so I wanted to see if this bias had also impacted image generatioon.
